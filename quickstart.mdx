## Prerequisites

Before you begin, make sure you have one of the following installed:

- [Bun](https://bun.sh) (recommended) - A fast JavaScript runtime
- [Node.js](https://nodejs.org) (v16 or higher) with npm, pnpm, or yarn

## Installation

Get started with the Cursor UI Clone by following these simple steps:

<Steps>
  <Step title="Clone the repository">
    First, clone the project to your local machine:
    
    ```bash
    git clone <your-repo-url>
    cd cursor-clone
    ```
  </Step>

  <Step title="Install dependencies">
    Install the project dependencies using your preferred package manager:
    
    <CodeGroup>
    ```bash Bun
    bun install
    ```

    ```bash npm
    npm install
    ```

    ```bash pnpm
    pnpm install
    ```

    ```bash yarn
    yarn install
    ```
    </CodeGroup>

    <Note>
      **Bun is recommended** for faster installation and development. It's a modern JavaScript runtime that's significantly faster than Node.js for most operations.
    </Note>
  </Step>

  <Step title="Start the development server">
    Launch the development server:
    
    <CodeGroup>
    ```bash Bun
    bun run dev
    ```

    ```bash npm
    npm run dev
    ```

    ```bash pnpm
    pnpm dev
    ```

    ```bash yarn
    yarn dev
    ```
    </CodeGroup>

    The site will be available at `http://localhost:5173`
  </Step>

  <Step title="Open in browser">
    Navigate to [http://localhost:5173](http://localhost:5173) in your browser to see the Cursor UI Clone in action!
  </Step>
</Steps>

### Development

Run the development server with hot module replacement:

```bash
bun run dev
```

### Build for Production

Create an optimized production build:

```bash
bun run build
```

The built files will be output to the `dist/` directory.

### Preview Production Build

Preview the production build locally:

```bash
bun run preview
```

## Tech Stack

This project is intentionally simple and uses:

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox
- **Vite** - Development server and build tool (using Rolldown)
- **No frameworks** - Pure vanilla JavaScript (when needed)

<Note>
The project uses `rolldown-vite@7.2.5` as specified in the package.json overrides for optimized performance.
</Note>

## Troubleshooting

### Port already in use

If port 5173 is already in use, Vite will automatically try the next available port (5174, 5175, etc.).

### Dependencies not installing

Make sure you're using a compatible version of Node.js (v16 or higher) or Bun. Try clearing your package manager's cache:

```bash
# npm
npm cache clean --force

# pnpm
pnpm store prune

# yarn
yarn cache clean
```
