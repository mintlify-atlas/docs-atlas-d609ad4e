## Overview

The navigation header is a sticky top bar that provides site-wide navigation and call-to-action buttons. It features a three-section layout with logo, navigation links, and action buttons.

## Visual Description

The navigation bar sits at the top of the page with a dark background, featuring:
- **Left section**: Cursor logo and brand name
- **Center section**: Four navigation links (Product, Enterprise, Pricing, Resources)
- **Right section**: Two buttons (Sign in, Download)

The header remains visible when scrolling due to its sticky positioning.

## HTML Structure

```html
<header>
  <nav class="nav flex justify-between align-center">
    
    <div class="nav__left flex justify-center align-center">
      <img class="nav__left__logo" src="AVATAR_CIRCLE_2D_DARK.png" alt="Cursor Logo">
      <h2>CURSOR</h2>
    </div>
    
    <div class="nav__center">
      <a href="#">Product</a>
      <a href="#">Enterprise</a>
      <a href="#">Pricing</a>
      <a href="#">Resources</a>
    </div>
    
    <div class="nav__right">
      <button type="button" class="btn btn-primary">Sign in</button>
      <button type="button" class="btn btn-secondary">Download</button>
    </div>
    
  </nav>
</header>
```

## CSS Styling

### Header Container

```css
header {
  padding: 0 20px;
  position: sticky;
  top: 0;
  background-color: var(--color-theme-bg);
}

.nav {
  margin: 0 90px;
  height: 52px;
}
```

<Note>
The `position: sticky` and `top: 0` properties keep the header visible during scrolling.
</Note>

## Implementation Notes

<Warning>
The negative margins on the logo (`margin-left: -8px`) are used for precise visual alignment. Adjust these values if using a different logo.
</Warning>

- The header uses CSS custom properties for theming (e.g., `var(--color-theme-bg)`)
- Fixed height of 52px ensures consistent spacing
- Horizontal margins of 90px on the nav element create page-wide padding
- Button borders are fully rounded (22px border-radius) for pill-shaped appearance
