## Overview

The features section displays three key product capabilities in a grid layout. Each feature card contains a title, description, link, and unique visual content (model selector, animated GIF, or image).

## Visual Description

This section includes:
- A section heading: "Stay on the frontier"
- Three equal-width feature cards arranged horizontally
- Each card has text at the top and a custom content area below
- Card 1: Shows a list of AI models with a checkmark indicator
- Card 2: Displays an animated GIF demonstrating codebase understanding
- Card 3: Features a team photo

## Customization Options

### Adding New Features

To add a feature card:

```html
<div class="features__grid__card">
  <div class="features__grid__card__item1">
    <p>Your Feature Title</p>
    <p>Description of the feature benefits.</p>
    <div>
      <a href="#">Learn more ↗</a>
    </div>
  </div>
  <div class="features__grid__card__item2">
    <!-- Your custom content -->
  </div>
</div>
```

### Custom Background

The `.c1` modifier class applies a custom background to the first card's content area. Create similar modifiers for other cards:

```css
.features__grid__card__item2.c2 {
  background-color: #YOUR_COLOR;
}
```

<Note>
Card 2 uses `.c2` class but doesn't define a custom background in the source CSS.
</Note>
