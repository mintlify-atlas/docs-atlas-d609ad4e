## Overview

The card component displays feature information in a two-column layout with an image and text description. Cards alternate their layout direction to create visual interest.

## Visual Description

Each card contains:
- **Text section**: Feature heading, description, and a "Learn more" link
- **Image section**: A product screenshot or illustration

Odd-numbered cards display text on the left, even-numbered cards display text on the right, creating a zigzag pattern down the page.

## CSS Styling

### Container

```css
.container {
  margin: 0px 90px 60px 90px;
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  gap: 100px;
  align-items: center;
}
```

<Note>
The 100px gap creates substantial vertical spacing between cards.
</Note>

### Card Base

```css
.card {
  width: 100%;
  display: flex;
  background-color: var(--color-theme-card-01-hex);
  padding: 20px;
}
```

### Alternating Layout

```css
.card:nth-child(odd) {
  flex-direction: row;
}

.card:nth-child(even) {
  flex-direction: row-reverse;
}
```

<Tip>
Using `:nth-child()` selectors automatically alternates the layout without additional classes.
</Tip>

### Text Section

```css
.card__item1 {
  text-align: start;
  width: 410px;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: start;
  padding: 20px;
}

.card__item1 h3 {
  font-size: 22px;
  font-weight: lighter;
}

.card__item1 h4 {
  font-size: 22px;
  font-weight: lighter;
  color: color-mix(in oklab, var(--color-theme-fg) 60%, transparent);
}

.card__item1 a {
  font-size: 16px;
  text-decoration: none;
  color: #f54e00;
}

.card__item1 a:hover {
  color: #cf4200;
}
```

The text section uses:
- Fixed width (410px) for consistent proportions
- Vertical centering with `justify-content: center`
- Lighter font weight for an elegant look
- Reduced opacity for the description text (60%)
- Orange accent color for links

### Image Section

```css
.card__item2 {
  flex: 1;
}

.card__item2__img {
  width: 100%;
}
```

<Warning>
The image section uses `flex: 1` to fill remaining space, making the layout responsive to different content widths.
</Warning>

## Complete Example

Here's a full working example with all styles:

```html
<section class="container p-20">
  <div class="card">
    <div class="card__item1">
      <h3>Your Feature Title</h3>
      <h4>A descriptive subtitle explaining the feature's benefits and capabilities.</h4>
      <a href="#">Learn more →</a>
    </div>
    <div class="card__item2">
      <img class="card__item2__img" src="feature-image.png" alt="Feature screenshot">
    </div>
  </div>
</section>
```

## Layout Mechanics

<Steps>
  <Step title="Container sets up vertical flow">
    `.container` uses flexbox with `flex-direction: column` and 100px gaps
  </Step>
  <Step title="Cards alternate direction">
    Odd cards use `flex-direction: row`, even cards use `row-reverse`
  </Step>
  <Step title="Text has fixed width">
    `.card__item1` maintains 410px width for consistent proportions
  </Step>
  <Step title="Image fills remaining space">
    `.card__item2` uses `flex: 1` to expand
  </Step>
</Steps>
