## Overview

The footer provides comprehensive site navigation, theme controls, language selection, and legal information. It uses a multi-column layout with organized link groups and interactive controls.

## Visual Description

The footer contains:
- Five columns of navigation links (Product, Resources, Company, Legal, Connect)
- A bottom bar with copyright, SOC 2 certification, theme switcher, and language selector
- Dark background matching the site's overall theme
- Organized information architecture for easy scanning

## Theme Switcher Component

The theme switcher includes three options:

| Theme | Icon | Data Attribute |
|-------|------|----------------|
| System | Monitor | `data-theme="system"` |
| Light | Sun | `data-theme="light"` |
| Dark | Moon | `data-theme="dark"` |

<Warning>
Theme switching functionality requires JavaScript to toggle the `.active` class and apply theme changes. The HTML/CSS only provides the UI structure.
</Warning>

## SVG Icons

The footer uses inline SVG icons for:
- Theme options (system, light, dark)
- Language selector (globe icon)

All icons use `currentColor` for fill/stroke, allowing them to inherit the button's text color:

```html
<svg viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">
  <path d="..." fill="currentColor" />
</svg>
```


### Active State

The currently selected theme has enhanced styling:

```css
.theme-btn.active {

  color: #ffffff;
}
```

## Accessibility Considerations

<Accordion title="Improving accessibility">
1. Add `title` attributes to theme buttons (already present)
2. Add `aria-label` to the language button
3. Make navigation links actual `<a>` tags instead of plain `<li>`
4. Add `role="navigation"` to the nav element
5. Consider adding `aria-current="page"` to active links
</Accordion>
