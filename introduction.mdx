## Overview

This project is a complete recreation of the Cursor.com homepage using pure vanilla HTML and CSS. It demonstrates modern web development techniques without relying on any frameworks, showcasing the power and flexibility of fundamental web technologies.

## Key Features

This clone includes all the major components and features of the original Cursor homepage:

- **Responsive Navigation** - Sticky header with navigation links and CTAs
- **Hero Section** - Eye-catching hero with headline and download button
- **Feature Cards** - Modular card components showcasing key features
- **Testimonials Grid** - Professional testimonials with avatars and company info
- **Feature Highlights** - Interactive feature showcase with images
- **Changelog Section** - Update timeline with version numbers
- **Footer** - Complete footer with links, theme toggle, and language selector

## Design System

The project implements a comprehensive design system with:

- **CSS Variables** for consistent theming across all components
- **Custom Typography** using the Cursor Gothic font
- **Flexible Layout** utilities with flexbox and grid
- **Dark Theme** optimized for professional developer aesthetics
- **Reusable Components** with modular, maintainable CSS

## Why This Project?

This clone serves multiple purposes:

1. **Learning Resource** - Study modern HTML/CSS techniques and responsive design patterns
2. **Template** - Use as a starting point for similar landing pages
3. **Portfolio Piece** - Demonstrate front-end development skills
4. **Educational Tool** - Understand how professional landing pages are structured
