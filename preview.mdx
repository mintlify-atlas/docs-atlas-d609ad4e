## Overview

The Cursor UI Clone is a pixel-perfect recreation of the Cursor.com homepage, built entirely with vanilla HTML and CSS. It showcases modern web design techniques without relying on any JavaScript frameworks.

## Design Highlights

### Color Scheme

The project uses CSS custom properties for consistent theming:

```css
:root {
    --color-theme-bg: #14120b;
    --color-theme-fg: #edecec;
    --color-theme-fg-02: #d7d6d5;
    --color-theme-card-hex: #1b1913;
    --color-theme-card-01-hex: #1d1b15;
    --color-theme-card-02-hex: #201e18;
    --color-theme-card-03-hex: #26241e;
    --color-theme-card-04-hex: #2b2923;
    --color-accent: #f54e00;
}
```

### Typography

The clone uses the authentic Cursor Gothic font:

```css
@font-face {
    font-family: "Gothic";
    src: url("./fonts/Gothic.woff2") format("woff2");
    font-weight: normal;
    font-style: normal;
    font-display: swap;
}
```

<Note>
  The Cursor Gothic font is sourced from [imjac0b/CursorGothic](https://github.com/imjac0b/CursorGothic)
</Note>

### Responsive Layout

The layout uses modern CSS techniques:
- **Flexbox** for navigation and card layouts
- **CSS Grid** for testimonials, features, and changelog sections
- **Custom properties** for consistent spacing and colors
- **Color-mix()** for semi-transparent text variations

```css
.color-soft {
    color: color-mix(in oklab, var(--color-theme-fg) 60%, transparent);
}
```

## Interactive Elements

### Buttons

Two button styles are implemented:

**Primary Button:**
```css
.btn-primary {
    background: var(--color-theme-bg);
    color: var(--color-theme-fg);
    border: 1px solid var(--color-theme-card-04-hex);
}

.btn-primary:hover {
    background: var(--color-theme-card-04-hex);
}
```

**Secondary Button:**
```css
.btn-secondary {
    background: var(--color-theme-fg);
    color: var(--color-theme-bg);
}

.btn-secondary:hover {
    background: var(--color-theme-fg-02);
}
```

### Hover Effects

Subtle hover effects throughout:
- Navigation links fade on hover
- Cards highlight with background color changes
- Buttons transition smoothly
- Links change to a darker orange shade

```css
.card__item1 a:hover {
    color: #cf4200; /* Darker orange */
}
```

## Technical Implementation

### Semantic HTML

The project uses semantic HTML5 elements:
- `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- Proper heading hierarchy (h1, h2, h3, h4)
- Accessibility-friendly structure

### Modern CSS

Leverages cutting-edge CSS features:
- CSS Grid and Flexbox for layouts
- Custom properties (CSS variables)
- `color-mix()` for dynamic colors
- `:nth-child()` selectors for alternating layouts
- Smooth scroll behavior

### Performance

Optimized for performance:
- Vite with Rolldown for fast builds
- Font preloading with `font-display: swap`
- Minimal dependencies (TypeScript only for dev)
- No JavaScript runtime overhead

## Browser Compatibility

The clone works in all modern browsers that support:
- CSS Grid
- CSS Custom Properties
- Flexbox
- `color-mix()` function

<Note>
  For best results, use the latest version of Chrome, Firefox, Safari, or Edge.
</Note>
